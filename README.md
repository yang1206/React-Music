# React Music

### 介绍
🚀🚀🚀 React Music，基于 React18、React-Router v6、React-Hooks、Redux、Redux Toolkit、TypeScript、Vite3、Ant-Design 的开源音乐APP，仅供学习。

## Project setup
```
pnpm install
```

### Compiles and hot-reloads for development
```
pnpm dev
```

### Compiles and minifies for production
```
pnpm  build
```
